# Maza BarberShop

Este é o repositório do projeto Maza Barbershop, um site elegante e moderno desenvolvido para uma barbearia real. Este projeto foi criado como um trabalho freelance, utilizando HTML, CSS (com SCSS para uma organização mais eficiente do código) e JavaScript.

## Stack utilizada

**Front-end:** HTML, CSS (com SCSS) e JavaScript



## Apêndice

A Barbearia Maza foi desenvolvida utilizando JavaScript para funcionalidades dinâmicas e SCSS para uma estilização organizada. A landing page foi estruturada nas seguintes seções:

- Home: Apresentação inicial destacando a identidade e os valores da barbearia.
- Sobre: Informações detalhadas sobre a história, missão e equipe da Maza Barbershop.
- Serviços: Galeria organizada que exibe os serviços oferecidos pela barbearia, com descrições e imagens ilustrativas.
- Venha conhecer: Convite para os clientes visitarem a barbearia, com informações de contato e um botão direcionando para agendamento de horários via WhatsApp.

Essas seções foram cuidadosamente planejadas e implementadas para oferecer uma experiência informativa e interativa aos visitantes do site da Maza Barbershop.

## Melhorias

- Consistência Visual: Mantenha um estilo visual consistente em todo o site. Use a mesma paleta de cores, fontes e estilos de botões.
  
- Tipografia: Garanta que a tipografia seja legível e apropriada para a identidade da marca. Utilize diferentes pesos de fonte para criar hierarquia e facilitar a leitura.
  
- Botões de Chamada para Ação (CTA): Certifique-se de que os botões de "Agendar horário" sejam visíveis e facilmente clicáveis em dispositivos móveis e desktop.
  
- Navegação: Considere adicionar um menu fixo no topo da página para facilitar a navegação. Os visitantes devem conseguir acessar diferentes seções do site sem precisar rolar muito.

- Feedback Visual: Adicione animações ou mudanças de cor nos botões ao passar o mouse, para melhorar a interação do usuário.
  
- Descrição dos Serviços: Adicione descrições mais detalhadas para cada serviço listado, para que os clientes saibam exatamente o que esperar.
  
- Galeria de Imagens: Considere adicionar uma galeria de imagens de alta qualidade mostrando diferentes estilos de cortes e serviços oferecidos.
  
- Testemunhos: Adicione uma seção de depoimentos de clientes satisfeitos para aumentar a credibilidade.
  
- Agendamento Online: Se ainda não tiver, implemente um sistema de agendamento online funcional, com confirmação por e-mail ou SMS.
  
- Integração com Redes Sociais: Aumente a integração com redes sociais para facilitar o compartilhamento e aumentar a visibilidade.
  
- Otimização de Imagens: Assegure-se de que as imagens estejam otimizadas para web para reduzir o tempo de carregamento da página.
  
- Design Responsivo: Verifique se o site está totalmente responsivo e se adapta bem a diferentes tamanhos de tela, especialmente dispositivos móveis.
  
- Acessibilidade: Garanta que o site seja acessível para todos os usuários, incluindo aqueles com deficiências. Utilize atributos ALT nas imagens e certifique-se de que a navegação pelo teclado esteja funcional.
  
- Meta Tags: Adicione meta tags apropriadas para melhorar a otimização para motores de busca (SEO).
  
- Conteúdo Relevante: Mantenha o conteúdo atualizado e relevante para atrair tráfego orgânico.
